# Practice Dashboard

Track your daily coding practice sessions.

## Deploy to Vercel

1. Push this folder to a GitHub repo
2. Go to vercel.com → New Project → Import your repo
3. Vercel auto-detects Vite → click Deploy
4. Done! 🎉

## Local Development

```bash
npm install
npm run dev
```

## Features
- Add practice days with links
- Click a day to open the session
- Mark days as done (hover → click "done")
- Delete days (hover → click "del")
- Streak & stats tracking
- Data saved in localStorage
